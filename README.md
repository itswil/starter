# Starter

A minimal starter template for building AI agent projects, created by and configured for AI agents.

## Stack

- React 19
- TypeScript 6
- Vite
- TailwindCSS v4
- oxlint + oxfmt
- Vitest
- Husky pre-commit, pre-push hooks

## Download Template

Run one of the following:

```bash
npx giget gh:itswil/starter my-new-project
pnpm dlx giget gh:itswil/starter my-new-project
bunx giget gh:itswil/starter my-new-project
# nobody uses yarn
```

## Getting Started (Local Development)

```bash
pnpm i
pnpm dev
```

## Commands

```bash
pnpm dev      # Start dev server
pnpm build    # Build for production
pnpm fmt      # Format code
pnpm lint:fix # Fix lint issues
pnpm test     # Run tests
```
